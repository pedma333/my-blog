# 의료 전문가의 마크다운 블로그를 시작합니다 🩺

안녕하세요. 마상혁입니다.

저는 **소아청소년과 전문의**이자 **바이오헬스 벤처 대표**, 그리고 **의학 칼럼니스트**로 활동하고 있습니다. 이 블로그는 제 의료 경험과 전문 지식을 나누고, 근거 기반의 건강 정보를 제공하기 위해 개설했습니다.

## 이 블로그에서 다룰 주제들

### 👨‍⚕️ 소아 건강
- **소아 성장**: 성장호르몬, 성조숙증, 성장 장애
- **소아 면역**: 백신, 감염 질환, 질병 예방
- **소아 영양**: 생애 초기 영양, 마이크로바이옴
- **소아 비만**: 대사 건강, 비만 치료

### 🔬 의료 과학
- **감염병**: 감염병 전문가 시각의 최신 정보
- **백신**: 백신의 중요성과 오해 바로잡기
- **노화 예방**: 생애주기 건강 관리
- **뇌 건강**: 뇌 발달과 인지 기능

### 💼 벤처 & 사업
- **바이오헬스**: 건강기능식품 개발 경험담
- **스타트업**: 의료 벤처 창업 노하우
- **제주 의료 관광**: 메디컬 투어리즘 전략

### 📰 의료 정책
- **의료 제도**: 의료계 정책 평론
- **근거 기반 의료**: Evidence-based medicine의 중요성

## 왜 마크다운 블로그인가?

기술적 간결함을 원했습니다:
- ✅ **순수 마크다운**: 복잡한 UI 없이 콘텐츠에 집중
- ✅ **자동화된 배포**: Git 한 줄로 전 세계에 배포
- ✅ **RSS 피드**: 구독자가 최신 글을 자동으로 받을 수 있음
- ✅ **버전 관리**: 모든 포스트를 Git으로 관리

## 이 블로그의 특징

### 📱 **모바일 친화적**
모든 기기에서 편하게 읽을 수 있습니다.

### 🌓 **다크모드 지원**
밤에도 눈이 편한 다크모드를 제공합니다.

### 📡 **RSS 피드**
구독 주소: `https://pedma333.github.io/my-blog/feed.xml`
- Feedly, Inoreader 등 RSS Reader에 등록 가능
- 네이버 블로그에 자동 연동

### 🚀 **완전 자동화**
- 포스트 작성 → 자동 배포
- GitHub Actions으로 RSS 피드 자동 생성
- 0초의 수동 배포 작업

## 기본 원칙

### 1️⃣ **근거 기반 (Evidence-Based)**
모든 의료 정보는 최신 논문, 가이드라인, 임상 경험에 기반합니다.

### 2️⃣ **명확한 커뮤니케이션**
- 의료진용과 일반인용을 구분
- 전문 용어는 괄호로 설명
- 복잡한 개념을 단순하게

### 3️⃣ **투명성**
- 의학적 근거 명시 (출처, 연도)
- 불확실한 것은 명확히 표현
- 이해상충이 있으면 공개
